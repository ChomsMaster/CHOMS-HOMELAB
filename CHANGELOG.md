## Version 0.1.0 - Initial Infrastructure

### Added

* Debian 13 installation
* Docker and Docker Compose
* WireGuard VPN
* Pi-hole DNS filtering
* PostgreSQL 17
* Nginx web server
* UFW firewall
* Fail2ban
* GitHub repository
* Initial project documentation

Initial production-ready infrastructure completed.
