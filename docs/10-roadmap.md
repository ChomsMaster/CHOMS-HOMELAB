# Project Roadmap

## Vision

CHOMS-HOMELAB is intended to become a complete self-hosted infrastructure platform capable of hosting production-ready services while serving as a continuous learning environment for infrastructure engineering, backend development and systems administration.

The project evolves through clearly defined milestones.

---

# Milestone 1 — Foundation Infrastructure

**Status:** ✅ Completed

### Objectives

* Debian installation
* Docker environment
* WireGuard VPN
* Pi-hole deployment
* PostgreSQL
* Nginx
* UFW
* Fail2ban
* GitHub repository
* Initial documentation

---

# Milestone 2 — Public Web Platform

**Status:** 🚧 In Progress

### Objectives

* Purchase domain
* Configure DNS
* Reverse Proxy
* HTTPS certificates
* Deploy CHOMS Master website
* Dynamic DNS
* Public availability
