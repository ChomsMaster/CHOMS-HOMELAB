# CHOMS Homelab

Servidor doméstico autogestionado para VPN, hosting web, DNS filtering y laboratorio backend.

## Stack actual

- Debian 13
- Docker
- Docker Compose
- Nginx
- WireGuard
- Pi-hole
- UFW
- Fail2ban
